****************************************************
** コンピュータ名の変更方法 (20010年03月17日改定)***
****************************************************
注意：ネットワーク接続の場合ネットワーク上に同じコンピュータ名が存在すると
      誤動作しますのでネットワークからはずして下さい。
注意：管理者(administrators)権限のあるログイン名でログインして下さい。
★評価版の場合コンピュータ名を"I1"又は"I2"にしないとインストールしても動作しません。

1.WindosVista,2008の場合
  (1)スタートボタンをクリックします。
  (2)コンピュータをクリックします。
  (3)上の左から３番目「システムのプロパティ」をクリックします。
  (4)システムの詳細設定をクリックします。
  (5)ユーザーアカウント制御で続行をクリックします。
  (6)コンピューター名タグをクリックします。
  (7)変更をクリックします。
  (8)コンピュータ名を"I1"又は"I2"(半角)にします。
  (9)OKをクリックしコンピュータを再起動します。

2.WindowsXPの場合
  (1)スタートボタンをクリックします。
  (2)マイ コンピュータをクリックします。
  (3)左側のその他内のコントロールパネルをクリックします。
  (4)パフォーマンスとメンテナンスをクリックします。
  (5)システムをクリックします。
  (6)コンピューター名タグをクリックします。
  (7)変更をクリックします。
  (8)コンピュータ名を"I1"又は"I2"(半角)にします。
  (9)OKをクリックしコンピュータを再起動します。

3.Windows Server2003,7の場合
  未確認です、たぶんXPと同じと思います。
4.(参考）openSUSE（KDE)はネットワークで設定します。
[Start]−[Configer Desktop]-[Advanced]-[YaST]-[Network Devices]-[Nwtwork Settings]
の[Hostname/DNS]でHostnameを変更する。
