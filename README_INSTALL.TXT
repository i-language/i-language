******************************************************
*** I言語 インストールの 説明 (2010年１０月１９日改定)***
******************************************************
I言語とは：
I言語は誰でもプログラムを即座(インスタント)に作れるようにした道具(ツール)です。
リレーショナルデータベースにデータを格納しSQLとI言語でプログラムを作ります。

注意：インストール用ファイルを作らない場合は下記にする必要があります。
コンピューター名を"I1"又は"I2"にする必要があります。
セットアップ名は”I”です。

動作条件：
(1)OSはWindows7,Vista,XP,2008,2005,2003であること。
(ただし、Vista以外での動作確認はしてありませんが動くはずです）
(2)管理者(administrators)権限を持ったログイン名でログインできること。
(3).NET Framework3.5SP1(2.0以上であれば動きます）がインストールされている事。
（７と2008R2はインストール不要です）
インストールはマイクロソフトのホームページのダウンロード(リソース)、
ダウンロードセンターのキーワードに".NET Framework 3.5"と入れて検索後
"「.NET Framework 3.5 Service Pack 1（フルパケージ)」”
をダウンロードしインストール下さい。

インストール方法。
注意：全て管理者権限を持ったログイン名で実行して下さい。
(1)インストール用ファイルを作らない場合はコンピュータ名を"I1"又は"I2"に変更します。
   README1.TXTに変更方法があります。
(2)以下のどれか１個のデータベース（RDBMS）をインストールします。
（SQL  Serverで開発しているので、一番充実していますので、SQL  Serverを推奨します）
  ①  SQL Server2008(R2)ExpressEdition又はSQL Server2008(R2)をインストールします。
     README21.TXTにインストール方法があります。(M2008EX,M2008)
  ②  PostgreSQL9.0.?をインストールします。
     README23.TXTにインストール方法があります。(P90)
  ③ IBM  DB2 VER9.7をインストールします。
     README25.TXTにインストール方法があります。(I9)
  ④ Oracle10GExpressEdition又はOracle11Gをインストールします。
     README26.TXTにインストール方法があります。(O10EX,O11)
(3)I言語 SERVERをインストールします。
   README3.TXTにインストール方法があります。
